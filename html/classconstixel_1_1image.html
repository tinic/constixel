Core class of constixel. Holds a buffer of an image width a certain size and format. Typical use:

constixel::image<constixel::format_8bit, 640, 480> image;

inline

Convert the current instance into a byte stream formatted for embedded displays. This function will write chunk_length of data into dst and update chunk_index on each call. You should pass chunk_index = 0 at the beginnig of the sequence. Returns true of there is more data, or false if there is no data left. This function is typically used to drive interrupt driven DMA transfers.

Template Parameters
dst_formatThe desired data format.
Parameters
dstThe buffer the data will be written to.
chunk_sizeThe requested chunk size in bytes. This value has to be kept the same during a full conversion sequence.
chunk_actualThe actual amount of bytes which were written into ptr during this call of the function.
chunk_indexThis value will increment after each call of the function. Has to be set to 0 on the first call.
Returns
true if there is more data to convert, false if we reached the end.
